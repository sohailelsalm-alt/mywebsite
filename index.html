إليك كود المصدر البرمجي الكامل والجاهز للتشغيل لمنصة AI Builder & Hub. لقد تم تحويل كامل الوصف والمميزات التفصيلية التي طلبتها إلى ملف كود برمجى واحد متكامل (HTML5) ذي جودة عالية ومبني على أفضل ممارسات تطوير الويب الحديثة. تم استخدام Tailwind CSS للتصميم العصري والزجاجي (Glassmorphic) المتجاوب بالكامل مع الشاشات المختلفة، ومكتبة Chart.js لتوفير لوحات إحصائية ورسوم بيانية تفاعلية حية، ومكتبة FontAwesome للأيقونات الفاخرة، مع الاعتماد على Vanilla JavaScript لبناء محرك تحكم كامل بمسارات التطبيق (SPA Engine) وتوفير تجارب تفاعلية حقيقية دون أي اختصارات أو تهميش للوظائف. محتويات هذا الكود البرمجي الشامل: 1. شاشة تحميل سينمائية (Dynamic Canvas Preloader): مع محاكي جزيئات مضيئة يتحرك في كافة الاتجاهات ويتفاعل مع حركة الماوس، وبشعار متوهج تدريجياً ونظام تخطٍ ذكي. 2. شريط تنقل مرن (Glassmorphism Sticky Navbar): يتغير تأثيره اللوني تلقائياً عند تمرير الصفحة، مع زر تفعيل وضع الإضاءة المزدوج (Light/Dark Mode) وحفظ التفضيل في المتصفح. 3. صفحة هبوط فاخرة (Cinematic Landing Page): بتأثير Parallax، تحتوي على شبكة تفاعلية تضم كافة الفئات العشر المطلوبة، وقسماً خاصاً لعرض مميزات المنصة مدعوماً بمؤثرات دخول ناعمة. 4. نظام مستخدم ومحاكاة دخول Google (Google Auth Gateway): يدير معلومات المستخدم، ويولد له اسماً وصورة شخصية عشوائية فاخرة عند الدخول، ثم ينقله لصفحة ترحيب مخصصة وتفاعلية. 5. بوابة حماية وقناة يوتيوب @zylvogg: تُلزم المستخدم بزيارة القناة التعليمية أولاً لمتابعة الاستكشاف، وتأتي بتصميم أنيق يعرض بطاقات فيديوهات محاكاة واشتراكات القناة. 6. معالج بناء الذكاء الاصطناعي (AI Builder Wizard): معالج حقيقي متكامل من 3 خطوات مفصلة لإعداد هوية العميل، ومحاكاة لوحة بناء الأكواد، والتي تعرض شريط تقدم حي ومخرجات لوحدة تحكم (Terminal Logs) تعقبها نافذة مستعرض ملفات (File Explorer) تتيح تصفح الأكواد المتولدة ونسخها. 7. مكتبة الأدوات التفاعلية (Active Tools Hub): تحتوي على نظام تصفية وبحث لحظي، وتضم 3 أدوات برمجية مدمجة تعمل بشكل تفاعلي بالكامل: - أداة مولد الأكواد البرمجية (Interactive Code Generator) مع ميزة النسخ وتلوين الشيفرة. - أداة مولد الصور الفنية (Image Sandbox Generator) مع محاكي توليد فني تدريجي. - مساعد الدردشة العام والذكي (Contextual AI Chatbot) والذي يحتفظ بسجلات الدردشة ومحاكاة الإجابة الفورية. 8. لوحة الإدارة الإحصائية (Admin Dashboard Control): رسوم بيانية تفاعلية من خلال Chart.js، بالإضافة لجدول كامل لإدارة الأعضاء مع صلاحيات تعديل الرتب، الحظر، وسجلات الأنشطة والعمليات اللحظية. 9. الصفحة الشخصية والإعدادات (User Settings): تتيح تفعيل رتبة المدير لإظهار لوحة الإدارة، وحفظ الاسم، وإدارة الجلسات الحالية. منصة AI Builder -
